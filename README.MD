# FlowchartCreationMiya v1.3.1

複数のノード図形や多彩な機能に対応した、Python & PyQt6製の高機能フローチャート作成デスクトップアプリです。

## 新機能・安定性改善 (v1.3.1)
- **【安定性改善】ノード追加時のクラッシュを修正:** Undo/Redo機能と追尾プレビュー（ゴースト）機能の併用時に発生していたC++オブジェクトへのアクセス違反を修正し、ノードを追加した瞬間にソフトが落ちる不具合を完全に解消しました。
- **図形追加の追尾プレビュー（ゴースト）機能:** ツールバーで「処理」「分岐」などの追加する図形を選ぶと、マウスポインタ（カーソル）に選んだ図形の半透明なプレビューがくっついて動くようになりました。キャンバス上の**どこに図形が作成されるか、クリックする前に一目で分かる**ようになり、直感的な作図が可能です。

## 既存の強力な機能
- **Undo / Redo 機能:** 操作ミスをしても「元に戻す (Ctrl+Z)」「やり直し (Ctrl+Y)」が可能。
- **コピー＆ペースト:** 選択した図形や線をコピー (Ctrl+C) して複製 (Ctrl+V) できます。
- **グリッド背景:** キャンバスに方眼線（グリッド）が表示され、位置合わせが簡単です。
- **線のスタイル変更:** 「書式ツールバー」から、エッジの太さや線種（実線・破線・点線）を変更可能。
- **グループ化機能:** 複数の図形をグループ化 (Ctrl+G) してまとめて移動できます。
- **豊富な整列ツール:** メニューバーの「配置」から、複数アイテムの「左・右・上・下揃え」「中央揃え」「等間隔配置」が一瞬で行えます。
- **範囲選択:** ドラッグによる一括選択と、Ctrl+クリックによる複数選択。
- **JSON保存/読込:** 作図状態を完全な形でセーブ・ロード。
- **CAD連携:** ワンクリックでJw_cad形式としてクリップボードへコピー。ノード内への線のめり込みも自動カット（クリッピング）されます。
- **高度な印刷:** リアルタイムプレビュー付きの印刷と、SVG・DXF・PNG出力。
