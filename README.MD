# FlowchartCreationMiya Pro (v2.0.2)

世界標準の作図ツール（VisioやDraw.io）に匹敵する機能を備えた、Python & PyQt6製のプロ仕様フローチャート作成デスクトップアプリです。

## 🚀 プロ仕様の強力な新機能
- **【NEW】モダンUI＆テーマ切替:** `qtawesome` を採用した美しいベクターアイコンと、`pyqtdarktheme` によるモダンなライト/ダークテーマをサポート。メニューの「表示」から切り替えられます。
- **【NEW】自動階層レイアウト (NetworkX):** ごちゃごちゃに置かれたノードも、メニューの「配置」→「★自動階層レイアウト」をクリックするだけで、上から下へ一瞬で綺麗なツリー構造に自動整列されます。
- **【NEW】直角配線（Orthogonal Routing）:** 書式ツールバーの「ルート」設定を「直角」に変更すると、Draw.ioのようにカクカクと曲がる美しい線を引くことができます。
- **【NEW】仕様書の自動生成機能:** メニューの「ファイル」→「仕様書(Excel)生成」を実行すると、フローチャートの繋がりを解析し、各ステップの手順書としてExcelファイル (`.xlsx`) を自動生成します。
- **【NEW】Draw.io 互換 (XMLエクスポート/インポート):** 作成したフロー図を Draw.io 形式で書き出し、ブラウザ版でそのまま読み込んで共有・編集することが可能です（インポートも対応）。

## 既存の機能とバグフィックス
- **ペースト追従の安定化:** コピー（`Ctrl+C`）後、自動的にペーストモードへ移行し、マウスにアイテム群のプレビューが追従します。
- **グループ化対応のコピペ:** グループ化 (`Ctrl+G`) した状態のアイテムでも正常にコピー＆ペーストできるようになりました。
- **Undo / Redo 機能:** いつでも操作を取り消せます。
- **JSON完全保存:** 状態の保存と読み込み。
- **CAD連携:** ワンクリックでJw_cad形式としてクリップボードへコピー。ノードへの線のめり込みも完全カットされます。
- **高度なエクスポート:** 画像(PNG/JPEG)、SVG、DXF出力、リアルタイムプレビュー付きの印刷機能。
