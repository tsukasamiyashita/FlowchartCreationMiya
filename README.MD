# FlowchartCreationMiya v1.2.0

複数のノード図形に対応し、JSON形式で保存・読み込みが可能なフローチャート作成デスクトップアプリです。
線の折り曲げや複数行テキストのサポートに加え、シームレスなズーム（拡大・縮小）機能が搭載されています。

## 新機能・UI改善 (v1.2.0 アップデート)
- **【NEW】範囲選択（ラバーバンド）機能:** 「選択」モード時にキャンバスの空きスペースをドラッグすることで、四角い範囲で複数のアイテムを一括選択できるようになりました。
- **【NEW】Ctrlキー複数選択:** `Ctrl` キーを押しながらアイテムをクリックすることで、個別に選択したり、選択を解除したりできます。
- **【NEW】複数アイテムの一括編集機能:**
  - **一括色変更:** 複数のノードを選択した状態でツールバーの「🎨 背景色」や「🔠 文字色」を押すと、選択したすべてのノードへ色が一度に適用されます。
  - **一括整列（配置メニュー）:** 上部メニューバーの「配置(A)」から、選択した複数のノードを「左に揃える」「上に揃える」ことができ、美しい図面作りをサポートします。
  - **一括削除:** 複数のアイテムを選択して「🗑️ 削除」ボタン、または `Delete` / `Backspace` キーを押すことで、依存する線なども含めて安全に一括削除できます。
- **UIレイアウトの維持:** 元の操作感を損なわないよう、描画ツールの配置などはそのままに、新しい機能を自然な形でメニューやマウス挙動に組み込みました。
- **出力時の「線の入り込み」を完全修正:** DXF出力やJw_cadへのコピー時、線がノード図形の中まで貫通してしまう問題を修正し、輪郭で正確にクリッピングされるようになりました。
