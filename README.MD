# FlowchartCreationMiya v1.1.0

複数のノード図形に対応し、JSON形式で保存・読み込みが可能なフローチャート作成デスクトップアプリです。
線の折り曲げや複数行テキストのサポートに加え、シームレスなズーム（拡大・縮小）機能が搭載されています。

## 新機能・UI改善
- **【NEW】タイトルとバージョンの更新:** アプリケーション名を「FlowchartCreationMiya」、バージョンを「v1.1.0」に変更しました。
- **【NEW】保存完了メッセージの表示:** 「上書き保存」「名前を付けて保存」「エクスポート」の実行完了時に、画面上に完了したことを知らせるポップアップメッセージ（ダイアログ）が表示されるようになりました。
- **上書き保存と名前を付けて保存:** 一般的なアプリケーションと同様に、保存機能が「上書き保存」と「名前を付けて保存」に分離されました。現在開いているファイル名がウィンドウ上部のタイトルバーに表示されるため、作業状況がひと目でわかります。
- **ツールバーの最適化とシンプル化:** ツールバーに並んでいたファイル関連ボタンを削除し、すべて上部の「ファイル」メニューに集約しました。残った描画ツールのボタン幅を少し狭く均一に調整し、スッキリとした美しいレイアウトに仕上げました。
- **出力時の「線の入り込み」を完全修正:** DXF出力やJw_cadへのコピーを行う際、線（エッジ）がノード図形の中まで貫通してしまう問題を修正しました。線の端点を自動計算して図形の輪郭で正確に切り取る（クリッピングする）ため、CAD上で見ても重なりがない非常に美しいデータとして出力されます。
- **印刷機能の最適化 (黄色い丸の非表示対応):** 印刷プレビューおよび実際の印刷出力、さらにはPNG/JPG等の画像エクスポート時に、エッジの中間制御点（折り曲げ用の黄色い丸）が自動的に非表示になるように改善しました。
- **高度な印刷設定とプレビュー:** メニューから印刷を選ぶと専用ダイアログが開き、プレビューをリアルタイムで確認しながら「印刷範囲」や「スケール（自動調整 / 任意の倍率）」を自由に調整できます。用紙への「はみ出し」も防止されています。
- **Jw_cad一発コピー機能:** メニューの「Jw_cadへコピー」を押すと、Jw_cadが読み込める形式（JwcTemp）でクリップボードに記憶。Jw_cad上で「貼り付け（Ctrl+V）」するだけで一瞬で展開可能です。
- **エッジテキストの移動機能:** エッジ（線）に追加された文字を選択し、ドラッグして自由に移動できるようになりました。ノードを動かしても文字はオフセットを維持して追従します。
