# Flowchart Editor

複数のノード図形に対応し、JSON形式で保存・読み込みが可能なフローチャート作成デスクトップアプリです。
線の折り曲げや複数行テキストのサポートに加え、シームレスなズーム（拡大・縮小）機能が搭載されています。

## 追加・改善事項
- **ズーム（拡大・縮小）機能の追加:** 広大なフローチャートを快適に編集できるようにズーム機能を実装しました。
  - **ツールバー操作:** 「🔍 拡大」「🔍 縮小」「🔍 100%（リセット）」ボタンをクリックしてズームできます。
  - **マウス操作:** `Ctrl` キーを押しながらマウスホイールを回転させることで、**マウスカーソルがある位置を中心にして**滑らかにズームイン・ズームアウトが可能です。
