# Flowchart Editor v1.0.0

複数のノード図形に対応し、JSON形式で保存・読み込みが可能なフローチャート作成デスクトップアプリです。
線の折り曲げや複数行テキストのサポートに加え、シームレスなズーム（拡大・縮小）機能が搭載されています。

## 新機能・追加事項
- **ファイル参照先の固定:** 「保存」「読込」「エクスポート」の際、エラーが出ない範囲の先頭ディレクトリ（システムのルートディレクトリ等）が初期状態として開くようになりました。
- **モード切り替え機能:** 「👆 選択」ボタンを新設し、ツールバーの各アクション（ノード追加、エッジ接続）がトグル式で使いやすくなっています。
- **連続配置＆接続機能:** - ノードの図形ボタンを選択した状態でキャンバスをクリックすると、クリックした位置へ連続してノードを配置できます。
  - エッジ接続モードの際も一度繋いだ後で解除されず、連続して次の線を繋いでいくことが可能です。
- **ヘルプメニュー:** 使い方とバージョン情報（v1.0.0）がヘルプメニューから確認できます。
- **ズーム機能:**
  - ツールバー操作: 「🔍 拡大」「🔍 縮小」「🔍 100%（リセット）」
  - マウス操作: `Ctrl` + マウスホイールで、カーソル位置を中心とした滑らかなズーム操作が可能です。
