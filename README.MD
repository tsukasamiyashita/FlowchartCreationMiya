# Flowchart Editor v1.0.1

複数のノード図形に対応し、JSON形式で保存・読み込みが可能なフローチャート作成デスクトップアプリです。
線の折り曲げや複数行テキストのサポートに加え、シームレスなズーム（拡大・縮小）機能が搭載されています。

## 新機能・UI改善
- **【NEW】エッジテキストの移動機能:** エッジ（線）に追加された文字を選択し、ドラッグして自由に移動できるようになりました。ノードを動かしても文字はオフセットを維持して追従します。
- **【重要バグ修正完了】エッジ接続の安定化:** 連続してエッジを接続する際に発生していたアプリケーションのクラッシュ問題を修正しました。内部的なオブジェクト管理と描画更新の処理を見直し、完全に安定して動作するように改善しました。
- **アイテム選択の視覚的ハイライト:** Qtデフォルトの点線枠を廃止し、ノード・線・文字などの選択時に、モダンで鮮やかなブルーの太線でハイライトされるように改善しました。
- **ツールバーUIの刷新:** CSS（スタイルシート）を適用し、ボタンの文字サイズ・余白を拡大して押しやすくしました。マウスホバー時や選択中（アクティブ時）の背景色も変化します。
- **起動時の最大化:** アプリケーション起動時に、ウィンドウが自動で最大化された状態で開くようになりました。
- **ファイル参照先の固定:** 「保存」「読込」「エクスポート」の際、先頭のディレクトリ（システムのルートディレクトリ等）が初期状態として開くようになりました。
- **連続配置＆接続機能:** クリックした位置への連続ノード配置、一度繋いだ後も解除されない連続エッジ接続が可能です。
